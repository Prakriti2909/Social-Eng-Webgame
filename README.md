# Social Engineering Web Simulator

> **Status: In progress** — dashboard and reports module complete, simulation module in development

An educational web platform for learning to identify phishing attacks and social engineering tactics. Built full-stack with Next.js and MongoDB.

## What it does

- Interactive dashboard showing attack categories, threat types, and awareness progress
- Reports module with breakdowns of phishing techniques and real-world attack patterns
- Covers phishing identification, social engineering tactics, and common attack vectors
- Phishing simulation module in development — will let users interact with realistic attack scenarios in a controlled environment

## Tech

Next.js, MongoDB, REST APIs, Tailwind CSS, GitHub (branch-based workflow with PRs)

## How to run

```bash
git clone https://github.com/Prakriti2909/social-engineering-simulator
cd social-engineering-simulator
npm install
npm run dev
```
